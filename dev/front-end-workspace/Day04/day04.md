### javascript 이용(이미지 핸들링 ---> 데이터 핸들링)

+ 프로그래밍을 하려면 이벤트 모델을 통해서
  + 이벤트 소스, 이벤트 종류, 이벤트 핸들러
    + tag는 준비되어 있다.
    + handler는 함수로 연결해줘야 한다.
    + 중첩 표시가 가능하다.
